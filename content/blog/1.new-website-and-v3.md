We are thrilled to announce the launch of our new website along the beta of the upcoming v3 version of Litestar. This major update is the result of months of hard work and dedication from our team, and we can't wait to share it with you! We also revamped and improved the documentation for the v3 !

## Website

With the upcoming v3, we wanted to have a new website that would reflect evolution of the project and the ecosystem around the framework.

### Plugin Directory

We wired a page to list all the plugins available for the Litestar framework. It includes official plugins maintained by the Litestar team as well as plugins developped by the community.

#### Submit Your Plugin

If you want to develop or register your plugin, you can have it listed there by submitting your plugin:

1. Fork the [litestar-org/plugin-registry](https://github.com/litestar-org/plugin-registry) repository
2. Add your plugin information following our submission guidelines
3. Create a pull request with your plugin details

We will then merge our submission and you will be listed in the plugin page.

### Blog Section

We decided to add a blog to share some exciting news about the litestar framework and the ecosystem. We will be posting release announcements and updates.

### Next improvements

We have a ton of ideas for the website, not sure all the things we have in mind will make it one day, but here is a list of some of the things:

- Deployment guide (how to deploy your Litestar application in production)
- Tutorials (interactive tutorials to learn Litestar hands-on)
- Starter templates (boilerplates to kickstart your projects or create a plugin)
- Docker and containerization best practices
- Performance optimization tips

This section will help you take your Litestar applications from development to production with confidence.

Stay tuned for more updates as we approach the v3 release!

## Join our community

The Litestar project lives because of our community. Here's how you can get involved:

### Connect on Discord

Join our vibrant [Discord community](https://discord.gg/litestar) where you can:

- Get help with your projects
- Share your creations
- Discuss new features and ideas
- Connect with other Litestar developers

### Contribute to the project

Litestar is open source and welcomes contributors! Whether you're interested in:

- **Code contributions**: Bug fixes, new features, performance improvements
- **Documentation**: Writing guides, improving examples, fixing typos
- **Community support**: Helping others on Discord and GitHub
- **Plugin development**: Creating tools that extend Litestar's capabilities

Every contribution makes a difference. Check out our [contribution guide](https://docs.litestar.dev/latest/contribution-guide) to get started.

### Talk about Litestar

If you enjoy using Litestar, consider sharing your experience with others, you can:

- Write blog posts or tutorials about your projects
- Talk to others about your experience with Litestar

## Get started today

If you want to try out Litestar, you can install it using pip or uv (or any package manager of your choice):

```bash
pip install litestar
# or
uv add litestar
```

Visit our [documentation](https://docs.litestar.dev) for comprehensive guides, explore the [plugin directory](/plugins) for useful extensions, and join our [Discord community](https://discord.gg/litestar) to connect with fellow developers.

We can't wait to see what you'll build with Litestar!
