We are excited to announce the release of Litestar v3 after months of dedicated development and invaluable community feedback. The framework continues to evolve as the modern Python ASGI framework built for scale. The repository is available on GitHub at [litestar-org/litestar](https://github.com/litestar-org/litestar) under the [MIT](https://github.com/litestar-org/litestar/blob/main/LICENSE) license.

::tip
The complete documentation is available at https://docs.litestar.dev.
::

## What's New in v3

Litestar v3 introduces significant improvements across the framework, focusing on performance, developer experience, and ecosystem growth. This release represents our commitment to providing a modern, efficient, and developer-friendly ASGI framework for Python applications.

### Enhanced Performance

Version 3 brings substantial performance improvements through optimized routing, faster serialization, and reduced memory footprint. These enhancements ensure your applications can handle more requests with lower latency.

### Improved Developer Experience

We've refined the API design based on community feedback, making common patterns more intuitive while maintaining the flexibility that makes Litestar powerful. Enhanced type safety and better error messages help developers catch issues early in the development process.

### Growing Ecosystem

With over 10+ official and community plugins now available, the Litestar ecosystem continues to expand. From database integrations to frontend frameworks, there's a plugin to accelerate your development workflow.

## Plugin Ecosystem

The release of v3 coincides with the launch of our new plugin directory, featuring:

- **Database Plugins**: Advanced Alchemy, AsyncPG, Piccolo ORM, Oracle DB
- **Frontend Integration**: Vite, HTMX for modern web development
- **Development Tools**: Granian server integration
- **Testing Utilities**: Polyfactory for comprehensive test data generation

Each plugin is maintained by the Litestar team and community contributors, ensuring quality and compatibility.

## Migration Guide

Upgrading from v2 to v3 is straightforward for most applications. We've prepared comprehensive migration documentation covering:

- Breaking changes and their alternatives
- New feature adoption strategies
- Plugin compatibility updates
- Performance optimization tips

Visit our [migration guide](https://docs.litestar.dev/latest/migration/v3) for detailed instructions.

## Get Started

Ready to experience Litestar v3? Install it using your preferred package manager:

```bash
pip install litestar
# or
uv add litestar
```

Explore our [getting started guide](https://docs.litestar.dev/latest/getting-started) to build your first application, or browse the [plugin directory](/plugins) to discover tools that can accelerate your development.
